1.vue cli 安装

        $ npm install -g @vue/cli
        $ vue --version
        @vue/cli 4.5.15

2.vue项目构建

    $ vue create 项目名
    

    project-sample/
     ├ package.json
     ├ vue.config.js : vue和webpack的定义文件
     ├ babel.config.js : babel的定义文件
     ├ src : 存放代码
     |  ├ main.js : 主程序入口
     |  ├ App.vue : main.js中被调用，主画面
     |  ├ components : App.vue以外组件
     |  ├ assets : 静态资源
     |  ├ plugins : main以外插件
     ├ public : 静态文件，html
     ├ dist : $ npm run build 执行后，webpack就会在此文件夹下打包，然后放到服务

3.启动vue

        $ cd project-sample
        $ npm run serve

浏览器中输入localhost:8080，画面可以显示就行

4.安装BootStrapVue安装

    $ npm install vue bootstrap-vue bootstrap

    在main.js里面追加下面内容
    

    import { BootstrapVue, IconsPlugin } from 'bootstrap-vue'
    import 'bootstrap/dist/css/bootstrap.css'
    import 'bootstrap-vue/dist/bootstrap-vue.css'
    Vue.use(BootstrapVue)
    Vue.use(IconsPlugin)
